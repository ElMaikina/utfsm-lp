El programa fue escrito y probado desde Arch Linux
interpretado desde el terminal con el siguiente 
commando:

	'python grafospp.py'

Por ende, es probable que los warnings sean distintos
al probar el código desde PyCharms, VS Code o cualquier
otro IDE.

El programa funciona con el input entregado e Aula y no
debiese haber mayor problema para ejecutar el programa.
Las funciones están descritas con el formato pedido y
el struct Nodo que definí también está comentado de la
misma manera.

Por último, el archivo incluye un input.txt de ejemplo
que fue el que usé para toda la tarea.
